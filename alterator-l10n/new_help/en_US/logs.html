System logs

System logs contains events logged by operation system components. This information can help you to diagnose and solve current problems or to prevent potential system problems.

There are several logs, e.g.:

Security
All messages about user authentication, session opening and closing etc.
Kernel
Operation system kernel events. This information can help you to fix hardware problems.

All information is divided into pages. You can go forward and backward, change page size, directly access first and last pages.
