1С-ХроноГраф

Назначение модуля

Модуль предназначен для интеграции различных служб Linux-сервера с программой 1С-ХроноГраф.

В модуле предусмотрены следующие операции:

  • импорт учётных записей 1С-ХроноГраф в службу каталогов LDAP.

Панель управления модулем

На панели управления модулем доступны для конфигурации а основные параметры импорта учётных записей из 1С-ХроноГраф:

  • целевой домен LDAP, в который будет произведён импорт учётных записей;
  • домен Samba, в который, при желании, также может быть произведён импорт учётных записей;
  • признак сброса паролей UNIX;
  • признак сброса паролей Samba;
  • имя файла XML "отчёта о пользователях" 1С-ХроноГраф.

Кнопка "Обновить", расположенная рядом с полем для выбора домена LDAP, позволяет произвести повторный запрос с целью получения доступных на локальном LDAP сервере доменов.

Домен самба будет доступен для выбора только в том случае, если на LDAP сервере включена схема данных Samba и установлена сама служба Samba вместе с модулем управления. В числе вариантов выбора Samba-домена доступен вариант "не импортировать в Samba", позволяющий пропустить данную операцию при импорте данных.

Поскольку в настоящей версии модуля управления, в целях безопасности, не осуществляется импорт паролей пользователей, по умолчанию новые учётные записи будут заблокированы. Признак сброса пароля позволяет установить системное имя пользователя в качестве пароля. Для более гибкой настройки, управление паролями UNIX и Samba производится отдельно. Используйте данную возможность с осторожностью.

Для импорта учётных записей выберите XML файл "отчёта о пользователях" 1С-ХроноГраф и нажмите на кнопку "Продолжить".
