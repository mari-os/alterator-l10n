POP3/IMAP сервер

Сервер dovecot підтримує декілька протоколів отримання пошти ("IMAP4", "IMAP4 через TLS/SSL", "POP3" та "POP3 через TLS/SSL"), a також декілька протоколів автентифікації користувачів (за змовчуванням використовується "PLAIN" — звичайна автентифікація по імені та паролю).

З міркувань безпеки настійно рекомендується вмикати та використовувати TLS/SSL-версії протоколів отримання пошти.

Крім основної функції, dovecot може використовуватися в якості зовнішнього сервера автентифікації для postfix. Для задіяння даної можливості, ввімкніть параметр "Порт служби автентифікації для Postfix".
