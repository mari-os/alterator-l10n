Встановлення програм

Модуль призначено для встановлення, оновлення та видалення пакетів програм.

При встановлені та оновлені програм вам спочатку потрібно оновити дані з джерел, вказаних у модулі «Джерела для встановлення ПЗ». Натисніть кнопку «Оновити» знизу діалогу. При цьому буде оновлено перелік доступних програм та позначені пакети, які потребують оновлення. Пам'ятайте, що при використанні джерел, які розташовано в Інтернеті, вам знадобиться працююче підключення до Інтернет.

Перелік програм працює в трьох режимах перегляду, які обираються з випадаючого переліку зверху діалога:

  • Групи (режим по замовченню) — ліворуч відображається дерево груп пакетів, праворуч — перелік пакетів у групі.
  • Зміни — ліворуч відображаються стани, праворуч — перелік пакетів обраного стану. По замовченню в цьому режимі відображається стан «Буде збережено» та встановлені пакети, для яких доступні нові версії (вони позначаються зірочкою).
  • Пошук — в цьому режимі ви можете шукати пакети за ключовими словами. Для цього введіть у полі «Ключові слова» одне або декілька слів, які ви хочете знайти у назві або опису пакетів, та натисніть клавішу Enter. Якщо вказано декілька слів, буде відображено пакети, які містять всі вказані слова. Кожний новий пошук буде додано у перелік ліворуч, а праворуч буде відображено результат цього пошуку.

Для встановлення або оновлення пакету двічі клацніть мишею по його назві або натисніть клавішу Enter:

  • Якщо пакет не встановлено (білий квадрат), його буде встановлено.
  • Якщо є нова версія пакету (зелений квадрат із зіркою), його буде позначено для оновлення.
  • Якщо пакет встановлено, його буде позначено для видалення (зелений квадрат з хрестиком)

Якщо ви хочете скасувати вибір пакетів, натисніть кнопку «Скинути».

Для безпосереднього застосування обраних дій (встановлення, оновлення або видалення), натисніть кнопку «Застосувати».

Увага! При виборі пакетів також автоматично обираються залежні пакети, тому до натиснення на кнопку «Застосувати», перейдіть у режим перегляду «Змін» та перевірте, чи не видаляються потрібні вам пакети.
