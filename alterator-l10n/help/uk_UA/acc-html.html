Центр управління системою

Центр управління системою — інтегрований набір інструментів для управління різними компонентами системи. Модулі налаштувань згруповано за завданням. Оберіть потрібний модуль налаштувань у меню, яке розташоване ліворуч.

Практично кожний модуль налаштувань супроводжується вбудованою довідкою. Відкрити довідку можна, перейшовши за посиланням «Довідка» у верхній частині сторінки.

Реєстрація

Для початку роботи з Центром управління системою потрібно зареєструватися: ввести ім'я та пароль для доступу до інтерфейсу налаштувань.

Додатково на етапі реєстрації можна обрати мову інтерфейсу. По замовченню буде запропоновано мову, яку визначено налаштуваннями браузеру. Інтерфейс Центру управління системою перекладено на:

  • англійську (en);
  • російську (ru);
  • українську (uk).

Параметри доступу до модуля

Для кожного окремо взятого модуля можна налаштувати параметри доступу. При натиску на посиланні «Параметри доступу до модуля...», яке розташовано знизу під кожним модулем, відкривається інтерфейс, який дозволяє обрати або видалити користувачів, яким потрібно надати або заборонити доступ до окремого модуля. Таким чином при реєстрації від ім'я користувача у Центрі управління системою він зможе керувати тільки налаштуваннями, доступ до яких йому надано.

Налаштування

Після вдалого входу Центр управління системою можна налаштувати (кнопка «Налаштування») обравши один з режимів:

  • Основний режим;
  • Режим експерту.

Вибір режиму впливає на кількість модулів які відображаються ліворуч. У режимі експерту відображаються всі модулі, а у основному режимі тільки найбільш використовувані.

Вихід

Для виключення можливості несанкціонованого доступу у Центрі управління системою не забувайте виходити з нього по завершенню роботи, натиснувши на кнопку «Вийти» у правому верхньому куту.
