Загрузчик операционной системы

Загрузчик операционной системы — системное программное обеспечение, осуществляющее загрузку операционной системы после включения компьютера. В качестве загрузчика используется GRUB.

При установке загрузчика будет составлено меню загрузки, включающее найденные операционные системы на компьютере.

Вы должны указать место для установки загрузчика:

  • Жёсткий диск — установить загрузчик в MBR (Master Boot Record) жёсткого диска. В этом случае устанавливается основной загрузчик для найденных операционных систем. Рекомендуется в большинстве случаев.
  • Раздел Linux — установить загрузчик на раздел. Рекомендуется, если основным используется другой загрузчик.
  • Не устанавливать загрузчик — отказаться от установки загрузчика. Настройку и установку загрузчика можно выполнить позднее. Внимание! При выборе этого пункта для запуска установленной операционной системы потребуется достаточно высокая квалификация и потому этот пункт не рекомендуется для новичков.
