Клиент Bacula

В этом модуле вы можете настроить агента для централизованного резервного копирования. Для этого нужно включить службу клиента, придумать и указать пароль агента и имя Bacula Director.

Примечание: пароль клиента должен совпадать с паролем для этого клиента на сервере с Bacula Director (модуль «Клиенты» раздела «Сервер резервного копирования»).
