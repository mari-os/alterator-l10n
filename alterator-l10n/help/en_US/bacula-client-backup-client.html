Client's setup

You should install bacula client on each machine you want to backup and then register it on using this screen. For each client you must specify:

  • Client's name. This is an any symbolic name you want;
  • Client's address;
  • File set for backup. Initial file set depends on operation system profile that you've selected during client creation (Linux or Windows).

To made your life easier we've prepared a config file for each client. You can simply download it and apply to client's file daemon.
