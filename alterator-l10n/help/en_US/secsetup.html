Disable execution bits (SUID)

When enabled, SUID bits in all binary files, except for explicitly specified ones, are ignored throughout the entire system.

Exceptions are presented as a colon-separated list of included SUID binary files.

Interpreter blocking (script launch prevention)

This option blocks unauthorized interpreter usage for code execution directly from the command line.

The "Interpreters" field should contain a colon-separated list of restricted interpreters.

Disable deleting open files

This option disables deleting open files.

The "Directories" field should contain a colon-separated list of directories.
