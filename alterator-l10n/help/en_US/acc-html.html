System management center

System management center is an integrated toolset to manage various OS components. Management tools (modules) are grouped by task. You can select appropriate module from left menu.

Virtually every module has help. You can open help using reference on the top of the page.

User interface translated into several languages:

  • English (en);
  • Russian (ru);
  • Ukrainian (uk).
Interface language is automatically detected from your browser settings or you can select your native language manually by selecting of an appropriate reference on the top of the page.
