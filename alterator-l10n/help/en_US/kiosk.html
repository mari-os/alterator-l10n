Configuring Kiosk mode

When in Kiosk mode, the user can run only the programs that are explicitly allowed by the administrator.

To allow running specific applications, a corresponding profile from the Profiles list should be enabled.

The applications included in the profile are listed in the Profile contents window.

Also, Kiosk mode must be selected.

Available modes:

  • disabled
  • block unauthorized application launch by regular users only
  • block unauthorized application launch by all users, including root
