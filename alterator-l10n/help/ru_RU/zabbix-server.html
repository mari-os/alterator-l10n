Zabbix server - центр мониторинга

Данный модуль предназначен для проведения первоначальной настройки, которая включает в себя создание базы данных для хранения данных сервера мониторинга.

Обратите внимание: первоначальная настройка может занять значительное время.

После прохождения первоначальной настройки на странице модуля будет отображаться ссылка на веб-интерфейс системы мониторинга и состояние компонентов сервера мониторинга. Также здесь можно изменить некоторые параметры серверной части системы мониторинга.
