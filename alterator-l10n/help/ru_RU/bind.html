DNS-сервер

Этот модуль предназначен для управления параметрами DNS-сервера. Предназначение DNS-сервера — по запросу, содержащему доменное имя хоста (компьютера или другого сетевого устройства), сообщить IP-адрес или (в зависимости от запроса) другую информацию.

Новый домен

Поле для ввода имени домена, для которого DNS-сервер будет авторитетным. Это значит, что имена из этого домена будут обрабатываться самим DNS-сервером, без запросов к вышестоящим серверам.

Пример для ввода: firma.localnet

Домен
Выбор из списка авторитетных доменов. Именно к выбранному домену относятся нижеперечисленные параметры.
Сети
Отметьте сетевые интерфейсы, на которых DNS-сервер будет отвечать на запросы.
Хосты
Таблица для показа либо выбора для удаления хостов для текущего домена.
Хост, IP-адрес
Поля для ввода дополнительных хостов в домен.
