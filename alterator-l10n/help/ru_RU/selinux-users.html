Пользователи SELinux

Модуль «Пользователи SELinux» позволяет создать пользователей SELinux для системных пользователей и настроить для них уровни, категории и выбрать профили.

Для настройки параметров SELinux для системного пользователя, выберите его из выпадающего списка слева внизу и нажмите кнопку «Добавить пользователя». После этого выберите его в списке слева.

После этого можно настраивать у пользователя SELinux следующие параметры:

Максимальный и минимальный уровень
Ограничение выбираемых при запуске сеанса пользователя уровня SELinux
Категории пользователя
Выберите категории из списка «Все категории», затем включите те категории из выбранных, которые необходимо установить у выбранного пользователя
Профиль пользователя
Выбор предопределённого профиля пользователя (например, officer_u) с предопределённым набором уровней
