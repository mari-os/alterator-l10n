Локальные зеркала репозиториев

Используя этот модуль вы можете создать локальное зеркало репозиториев, перечисленных в столбце «Репозиторий». Это может быть полезно для дальнейшего использование этого зеркала как источника пакетов.

Зеркалирование включается опцией «Разрешить зеркалирование». При этом реальная загрузка данных происходит каждую ночь. Первая синхронизация с репозиторием может занять длительное время.

Модуль позволяет настроить доступ к созданному зеркалу посредством NFS. За включение этой возможности отвечает опция «Доступ к хранилищу через NFS».

Важно самостоятельно следить за наличием достаточного количества свободного места для создания зеркала. Текущее значение указывается в строке «Свободное место».
