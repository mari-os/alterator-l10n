Загрузчик операционной системы

Загрузчик операционной системы — системное программное обеспечение, осуществляющее загрузку операционной системы после включения компьютера. В качестве загрузчика используется GRUB.

При установке загрузчика будет составлено меню загрузки, включающее найденные операционные системы на компьютере.

Вы должны указать место для установки загрузчика:

  • EFI — установить загрузчик в специальный раздел EFI (при установке в режиме EFI). Есть варианты, зависящие от оборудования. Если не работает один вариант, попробуйте другие.
    • рекомендуемый — при установке загрузчика, в NVRAM будет добавлена запись, без которой большинство компьютеров не смогут загрузиться во вновь установленную операционную систему.
    • сначала очистить NVRAM — перед добавлением записи в NVRAM, её содержимое будет сохранено в /root/.install-log, после чего из неё будут удалены все загрузочные записи, что приведёт к восстановлению полностью заполненной NVRAM и гарантирует загрузку вновь установленной операционной системы.
    • запретить запись в NVRAM — выбирайте этот вариант только, если инсталлятор не может создать запись в NVRAM или если заведомо известно, что запись в NVRAM может вывести компьютер из строя (вероятно, запись в NVRAM придётся создать после установки операционной системы средствами BIOS Setup).
  • RAID — установить загрузчик в раздел созданного массива RAID.
  • Жёсткий диск — установить загрузчик в MBR (Master Boot Record) жёсткого диска. В этом случае устанавливается основной загрузчик для найденных операционных систем. Рекомендуется в большинстве случаев.
  • Раздел Linux — установить загрузчик на раздел. Рекомендуется, если основным используется другой загрузчик.
  • Не устанавливать загрузчик — отказаться от установки загрузчика. Настройку и установку загрузчика можно выполнить позднее. Внимание! При выборе этого пункта для запуска установленной операционной системы потребуется достаточно высокая квалификация и потому этот пункт не рекомендуется для новичков.

* Пароль на загрузчик по умолчанию устанавливается для пользователя boot.
