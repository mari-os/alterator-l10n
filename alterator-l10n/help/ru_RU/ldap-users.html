Пользователи

Данный модуль позволяет управлять учётными записями пользователей, централизованно хранящимися в LDAP.

Вы можете:

  • Добавить учётную запись нового пользователя;
  • Удалить существующую учётную запись;
  • Изменить параметры существующей учётной записи.

Использование модуля

Новая учётная запись
После ввода допустимого имени необходимо нажать Создать, после чего имя отобразится в списке слева. Для дополнительных настроек необходимо выделить добавленное имя либо, если необходимо изменить существующую учётную запись, выбрать её их списка.
Фамилия, Имя, Отчество
Поля для ввода информации о владельце учётной записи.
Пароль
Для каждой учётной записи создаётся пароль, который можно ввести вручную либо воспользоваться автогенерацией пароля (отметка Создать автоматически).
Организация, Должность, Телефон, Мобильный телефон
Поля для ввода дополнительной информации о владельце учётной записи.
Электронная почта
По умолчанию, для каждой создаваемой учётной записи создаётся e-mail вида имя_пользователя@домен. При необходимости вы можете добавить дополнительные адреса.
Домашний каталог
Каталог пользователя, в котором он будет иметь полные права. В случае регистрации в консоли работа начинается именно в этом каталоге. Обычно домашний каталог пользователя располагается в /home/имя_пользователя, где имя_пользователя — это имя учётной записи.
Интерпретатор команд
Это командная оболочка, запускаемая по умолчанию при регистрации пользователя в текстовой консоли. По умолчанию используется /bin/bash.

Пример использования

Для сотрудника Михаила Иванова необходимо создать учётную запись пользователя mivanov с паролем 3rY50Mm и прочими стандартными значениями учётной записи (домашний каталог, интерпретатор команд, и т.д.).

  • В поле «Новая учётная запись» введите mivanov
  • Нажмите «Создать»
  • Выберите в списке пользователей строку mivanov
  • В разделе «Учётная запись» в поля «Фамилия» и «Имя» введите Иванов и Михаил соответсвенно (эти записи не являются обязательными)
  • Здесь же в разделе «Учётная запись» введите пароль 3rY50Mm в поля, отмеченные как «введите фразу» и «повторите фразу»
  • Нажмите «Сохранить параметры»
