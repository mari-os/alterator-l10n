Модуль Alt Hardening

AltHa — это модуль (расширение) безопасности Linux, который в настоящее время имеет три варианта защиты пользовательского пространства:

  • игнорировать биты SUID в двоичных файлах (возможны исключения);
  • запретить запуск выбранных интерпретаторов в интерактивном режиме;
  • отключить возможность создания ссылок на открытый файл в выбранных каталогах.

Запрет бита исполнения (SUID)

При включенной опции, биты SUID во всех двоичных файлах, кроме явно перечисленных, игнорируются в масштабе всей системы.

Исключения это список включенных двоичных файлов SUID, разделенных двоеточиями.

Блокировка интерпретаторов (запрет запуска скриптов)

Опция блокирует несанкционированное использование интерпретатора для выполнения кода напрямую из командной строки.

Поле интерпретаторы должно содержать разделенный двоеточиями список ограниченных интерпретаторов.

Отключение возможности удаления открытых файлов

Опция отключает возможность удаления открытых файлов.

Поле директории должно содержать разделенный двоеточиями список каталогов.
